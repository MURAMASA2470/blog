Windows10 Version1803（2018年4月）から、コマンドラインで `curl` と `tar` が使えるようです。

```
>where curl
C:\Windows\System32\curl.exe

>where tar
C:\Windows\System32\tar.exe
```

もうだいぶ前の話なんですが、自分用のメモとして書いてみました。

ありがたいです。
