## テーブルの作成
次の SQL を実行して、テーブル `memo` を作成しておきます。

```
create table memo (
  id serial primary key,
  txt varchar(200) not null,
  updated timestamp not null default current_timestamp,
  created timestamp not null default current_timestamp
);
```
