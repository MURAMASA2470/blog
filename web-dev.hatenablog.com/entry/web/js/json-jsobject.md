JSON と JavaScript オブジェクト について、プロパティ名（キー）の違いを書いています。

## JSON
JSON のプロパティ名は、引用符で囲んで表現します。

```json
{ "name": "太郎", "age": 42, "isAdmin": false }
```

## JavaScript オブジェクト
引用符で囲んでも囲まなくても、どちらでも大丈夫なようです。

```json
{ "name": "太郎", "age": 42, "isAdmin": false }
```

```javascript
{ name: "太郎", age: 42, isAdmin: false }
```

上の２つはどちらも有効です。
